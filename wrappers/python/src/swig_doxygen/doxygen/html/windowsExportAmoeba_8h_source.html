00005  * Shared libraries are messy in Visual Studio. We have to distinguish three

00015  *     OPENMM_AMOEBA_BUILDING_{SHARED|STATIC}_LIBRARY
00016  * Client code normally has no special symbol defined, in which case we'll
00017  * assume it wants to use the shared library. However, if the client defines
00018  * the symbol OPENMM_USE_STATIC_LIBRARIES we'll suppress the dllimport so
00019  * that the client code can be linked with static libraries. Note that
00020  * the client symbol is not library dependent, while the library symbols
00021  * affect only the OpenMM library, meaning that other libraries can
00022  * be clients of this one. However, we are assuming all-static or all-shared.
00023  */
